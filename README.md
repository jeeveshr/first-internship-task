# Grand Horizon Hotel - Web Development Project

## Overview
This is a premium, multi-section luxury hotel website built using **HTML5, CSS3, and Bootstrap 5**. It was created as part of my first internship task to demonstrate high-end web design and development skills.

## Features
- **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.
- **Premium Aesthetics**: Gold-themed design with smooth animations and glassmorphism effects.
- **Interactive Elements**: 
  - Dynamic Navbar (changes on scroll).
  - Hero section with floating particles.
  - Animated Stat counters.
  - Interactive Room and Amenity cards.
  - Fully functional Gallery and Testimonials sections.
  - Modern Booking Form.
- **Micro-animations**: Hover effects and scroll-linked animations for a premium feel.

## Technologies Used
- **HTML5**: Semantic structure.
- **CSS3**: Custom styling, external stylesheet, and CSS Variables.
- **Bootstrap 5**: Grid system, components (Navbar, Buttons, Forms), and utility classes.
- **Bootstrap Icons**: Scalable vector icons.
- **Google Fonts**: premium typography (Playfair Display & Inter).

## How to Run
Simply open `index.html` in any modern web browser.
